# M5Stack DIY Lab

A static inspiration site for M5Stack devices, CAPs and Units. Pick a device, toggle the parts you own, and browse 20 curated DIY projects that match — from "blink the LED" beginner builds to advanced off-grid LoRa builds.

## What's in here

- `index.html`, `styles.css`, `app.js` — the site. No framework, no build step, zero runtime dependencies.
- `data/products.json` — generated by the scraper below.
- `data/projects.json` — hand-curated project recipes.
- `images/{devices,caps,units}/<id>/*.webp` — product photos pulled from shop.m5stack.com.
- `scripts/scrape.mjs` — Node 18+ scraper. Uses built-in `fetch` only.
- `.github/workflows/pages.yml` — auto-deploys the site to GitHub Pages on push to `main`.

## View it locally

```sh
python3 -m http.server 8000
# open http://localhost:8000
```

## Refresh product data and images

```sh
node scripts/scrape.mjs
```

This re-downloads from shop.m5stack.com and regenerates `data/products.json`. Images that already exist on disk are skipped, so repeat runs are cheap.

## Add a new project

Edit `data/projects.json`. Each entry references `compatibleDevices`, `requiredParts`, and `optionalParts` using the stable IDs from `data/products.json` (for example `m5sticks3`, `cardputer-adv`, `cap-lora`, `unit-env-iii`).

## Deploy

In the repo's **Settings → Pages**, set *Source* to **GitHub Actions**. Push to `main` and the workflow in `.github/workflows/pages.yml` will publish the site.
